At this point, the Anvil configuration is complete.

Next, we need to flash the Anvil firmware onto the Pico W.

.. warning::

    Before reinstalling the firmware, make a backup of all important files stored on your Pico W.

Power off the Pico W, then push and hold the **BOOTSEL** button on the Pico W. Connect it to your computer using a Micro USB cable, and release **BOOTSEL** once the drive **RPI-RP2** appears on your computer.

.. image:: img/iot/RPI-RP2.webp

You can find a `.uf2` firmware file in the path ``Ultimate-Starter-Kit-for-Pico-W/Python/3.Firmware``. Drag and drop this `.uf2` file into the **RPI-RP2** drive. After a short while, you should see ``main.py`` and ``boot.py`` appear on the drive.

Select **MicroPython (Raspberry Pi Pico).COMXX** as the interpreter in Thonny IDE. After clicking **View -> Files**, you will see both the local drive and the drive of the Raspberry Pi Pico.

Modify the ``boot.py`` script to include your WiFi name (SSID) and password.

.. image:: img/iot/boot.png

Click **Run current script**.

Upload ``dht.py`` and ``lcd1602.py`` from the path ``kepler-kit-main/libs`` to the Raspberry Pi Pico W.

Replace ``main.py`` with the following code, and insert your Uplink key where indicated:

.. code-block:: python

    import anvil.pico
    import uasyncio as a
    from machine import Pin, I2C

    from lcd1602 import LCD
    i2c = I2C(1, sda=Pin(6), scl=Pin(7), freq=400000)
    lcd = LCD(i2c)

    from dht import DHT11
    sensor = DHT11(Pin(16, Pin.OUT, Pin.PULL_DOWN))

    UPLINK_KEY = "your_uplink_key"

    @anvil.pico.callable(is_async=True)
    async def show_message(text):
        print(f"show anvil's input message: {text}")
        lcd.clear()
        lcd.message(text)
        return

    @anvil.pico.callable(is_async=True)
    async def publish_data():
        sensor.measure()
        return "Temperature: {}, Humidity: {}".format(sensor.temperature, sensor.humidity)

    # Connect the Anvil Uplink. In MicroPython, this call will block forever.
    anvil.pico.connect(UPLINK_KEY)

    # There's lots more you can do with Anvil on your Pico W.
    # See `Anvil on the Pico W <https://anvil.works/pico>`_ for more information.
